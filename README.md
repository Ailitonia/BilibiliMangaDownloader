# BilibiliMangaDownloader
下载哔哩哔哩漫画到本地

## 需求python版本

**3.10+**

## 用法

1. 安装依赖: `pip install requirements.txt`
2. 在 `.env` 中配置你的哔哩哔哩用户 cookies (也可以不配置, 就只能下载免费章节)
3. 运行: `python download_bilibili_manga.py`
4. 按提示输入需要下载的漫画
